模型语法

方程表示

在lavaan中，模型是用一个回归方程组来表示。

myModel <- ' # 主要回归方程
             y1 + y2 ~ f1 + f2 + x1 + x2
                  f1 ~ f2 + f3
                  f2 ~ f3 + x1 + x2
             
             # 定义潜变量
             f1 =~ y1 + y2 + y3
             f2 =~ y4 + y5 + y6
             f3 =~ y7 + y8 + y9 + y10

             # 方差和协方差
             y1 ~~ y1
             y1 ~~ y2
             f1 ~~ f2
          
             # 截距项
             y1 ~ 1
             f1 ~ 1

如果模型很长，可以将模型(单引号之间的内容)储存入myModel.lav的txt文档中，用以下命令读取：

myModel <- readLines("/mydirectory/myModel.lav")  # 这里需要绝对路径

注意自变量可以是潜变量： y ~ f1 + f2 + x1 + x2 f1 ~ f2 + f3 f2 ~ f3 + x1 + x2 上面f开头的f1、f2就是潜变量：

如何定义潜变量

在lavaan中潜变量通过=~（measured by）来定义： f1 =~ y1 + y2 + y3 f2 =~ y4 + y5 + y6 f3 =~ y7 + y8 + y9 + y10``

方差与协方差

用~~表示变量之间的协方差或者方差： y1 ~~ y1 # 方差 y1 ~~ y2 # 协方差 f1 ~~ f2 # 协方差

截距

y1 ~ 1 f1 ~ 1
