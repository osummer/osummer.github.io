简介

parameters包主要用于不同模型的参数分析。主要函数：

  • parameters()
  • model_parameters()
  • select_parameters()
  • reduce_parameters()

主要功能

- [参数汇总](https://easystats.github.io/parameters/articles/model_parameters.html)
- [标准化参数](https://easystats.github.io/parameters/articles/model_parameters_standardized.html)
- [稳健估计](https://easystats.github.io/parameters/articles/model_parameters_robust.html)
- [数据缺失处理](https://easystats.github.io/parameters/articles/model_parameters_mice.html)
- [降维](https://easystats.github.io/parameters/articles/parameters_reduction.html)

- [面板数据分析](https://easystats.github.io/parameters/articles/demean.html)

示例

参数提取model_parameters

# 相关分析
cor.test(iris$Sepal.Length, iris$Sepal.Width) |>
  model_parameters()

# Bayes相关
BayesFactor::correlationBF(iris$Sepal.Length, iris$Sepal.Width) |>
  model_parameters()

# t检验
t.test(mpg ~ vs, data = mtcars) |>
  model_parameters()
  
# Bayes检验
BayesFactor::ttestBF(formula = mpg ~ vs, data = mtcars) |>
  model_parameters()

# ANOVA
aov(Sepal.Length ~ Species * Sepal.Width, data = iris) |>
  model_parameters(
    omega_squared = "partial",
    eta_squared = "partial",
    ci = .8)

aov(mpg ~ am + Error(gear), data = mtcars) |>
  model_parameters()

# 一般回归
model <- lm(Sepal.Width ~ Petal.Length * Species, data = iris)

## regular model parameters
model_parameters(model)

## standardized parameters
model_parameters(model, standardize = "refit")

## heteroscedasticity-consitent SE and CI
model_parameters(model, vcov = "HC3")

# GLM
glm(vs ~ poly(mpg, 2) + cyl, data = mtcars, family = binomial()) |>
  parameters(exponentiate = TRUE, summary = TRUE, ci_method = "wald")

# 混合模型
model <- lmer(Sepal.Width ~ Petal.Length + (1 | Species), data = iris)

## model parameters with CI, df and p-values based on Kenward-Roger approximation
model_parameters(model, ci_method = "kenward", effects = "fixed")

# 结构方程模型
library(psych)
model <- psych::fa(attitude, nfactors = 3)
model_parameters(model)

# PCA
psych::pca(mtcars, nfactors = 3) |>
  parameters()

# lavva
model <- lavaan::cfa(" visual  =~ x1 + x2 + x3
                       textual =~ x4 + x5 + x6
                       speed   =~ x7 + x8 + x9 ",
  data = HolzingerSwineford1939)
  
model_parameters(model)

参数筛选

lm(disp ~ ., data = mtcars) |>
  select_parameters() |>
  model_parameters()

降维

不同于筛选，降维是用有限的几个隐变量替代输入变量，隐变量与输入变量高度相关，PCA就是降维方法的一种：

data("attitude")
model <- lm(rating ~ ., data = attitude)
parameters(model)
newmodel <- reduce_parameters(model)
parameters(newmodel)
